Doomemacs Cheatsheet

Open files

Recently opened files SPC f r
Reload last session SPC q l
Open org-agenda SPC o A
Open project SPC p p
Jump to bookmark SPC RET
Open private configuration SPC f P
Open documentation SPC h d h

Jump to files, buffers or projects

Find bookmark SPC RET
Browse from current directory SPC f f, SPC .
Find file in project SPC p f, SPC SPC
Find recently opened file SPC f r
Open another project SPC p p
Switch to buffer in current workspace SPC b b, SPC ,
Switch to buffer SPC b B, SPC <

Code completion

Go to next candidate C-n
Go to previous candidate C-p
(evil) Go to next candidate C-j
(evil) Go to previous candidate C-k
Display documentation (if available) C-h
Move to previous page of candidates C-u
Move to next page of candidates C-d
Filter candidates C-s
Search candidates with helm/ivy C-S-s
Complete common C-SPC
Complete common or select next candidate TAB
Select previous candidate S-TAB

Jump-to navigation

Jump to file in project SPC p f, SPC SPC
Jump to file from current directory SPC f f, SPC .
Jump to symbol in file SPC s i
